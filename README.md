## 📝구현 설명📝

Open Weather Map의 API로 날씨 정보를 가져왔습니다.

select에서 선택한 지역의 실시간 날씨 정보가 화면에 보여집니다.

지역 이름, 날씨 아이콘, 현재 온도, 바람, 체감 온도를 확인할 수 있습니다.
